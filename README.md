## Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ferhateroglu/react-redux-realworld.io
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start server
   ```sh
   npm run start
   ```
